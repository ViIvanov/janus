Установка и начало работы

Перед установкой следует внимательно ознакомится с системными требованиями к продукту и установить недостающие компоненты, скачав их, например, с сайта Корпорации Майкрософт.

Дистрибутив продукта можно получить по адресу http://www.rsdn.ru/article/?353.

Для работы с продуктом потребуется учётная информация.

Рекомендуется заранее провести регистрацию на сайте RSDN для получения логина и пароля. Для регистрации понадобится работающий email. Также рекомедуется указывать минимальную контактную информацию: имя, фамилию, город и страну проживания.

Рекомендуется устанавливать stable версии продукта, однако некоторые beta версии продукта вполне пригодны для использования. По вопросу использования beta версий продукта можно проконсультироваться у разработчиков на  форуме продукта.

Новая установка

Распакуйте архив с дистрибутивом и запустите программу установки продукта. Укажите папку для установки продукта или оставте предложенную по умолчанию. После завершения работы программы установки запустите продукт.

При первом запуске на форме Настройки пользователя (можно привести скриншот из статьи) укажите папку, где будет хранится локальная База Данных сообщений и персональные настройки пользователя (рекомендуется указывать папку, отличную от папки для установки продукта), а также учётную информацию, полученную ранее. Подтвердите указанную информацию.

Теперь следует оформить подписку на интересующие вас форумы. Выберите меню Сервис - Подписка на форумы и в появившейся форме отметьте галочками интересующие вас элементы списка. Если список пуст, то следует его обновить, воспользовавшись кнопкой Синхронизировать список с сервером. (Здесь можно вставить второй скриншот из статьи.) Подтвердите указанную информацию. В дереве навигации появятся выбранные форумы.

Выберите пункт меню Сервис - Синхронизироваться с сервером. По завершении синхронизации обновится дерево навигации и список сообщений.

Программа готова к работе.

Обновление продукта

Перед обновлением продукта следует удалить предыдущую версию продукта с помощью Панели управления Windows.

Затем распакуйте архив с дистрибутивом и запустите программу установки продукта. Укажите папку для установки продукта или оставте предложенную по умолчанию. После завершения работы программы установки запустите продукт.

Программа готова к работе.

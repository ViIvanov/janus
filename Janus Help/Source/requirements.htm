Системные требования

Продукт разработан для операционной системы Microsoft Windows и тестировался в Windows XP/ Windows 2000 (можно ещё чего добавить, если кто ещё где работает.)

Для работы продукта требуется:

  1. Microsoft .NET Framework Runtime 1.1
  2. Microsoft Jet 4 OleDb Provider (MDAC 2.6 или выше)
  3. Для русских версий Windows 2000 high encryption pack

Данные компоненты можно скачать, например, с сайта Корпорации Майкрософт.

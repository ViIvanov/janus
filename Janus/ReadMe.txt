RSDN@Home
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

Описание
--------------------------------------------------
RSDN@Home (Janus) - клиент для офлайнового чтения форумов http://rsdn.ru/.
Задать вопросы и внести предложения можно на форуме проекта:
http://rsdn.ru/forum/group/janus.aspx


Системные требования
--------------------------------------------------
1.	.Net Framework 3.5 или более новый.
	Последнию версию .Net Framework можно скачать здесь:
	http://microsoft.com/net/Download.aspx

2.	Права администратора
